# 大富翁

大富翁遊戲的 Python 版。

---

## 專案結構

```
monopoly-python/
├── main.py                 # 遊戲進入點
├── core/                   # 核心遊戲邏輯（無 UI 依賴）
│   ├── dice.py             # 骰子
│   ├── land.py             # 地產模型
│   ├── player.py           # 玩家狀態與移動
│   ├── chance_card.py      # 機會卡（20 張）
│   ├── fate_card.py        # 命運卡（19 張）
│   ├── shop.py             # 商店道具
│   └── game_state.py       # 回合狀態引擎
├── ui/                     # Pygame 畫面層
│   ├── board_renderer.py   # 棋盤與玩家精靈
│   ├── info_panel.py       # 右側訊息欄
│   ├── dialog.py           # 確認對話框 / 道具欄
│   ├── shop_screen.py      # 商店視窗
│   ├── card_popup.py       # 機會 / 命運卡彈窗
│   └── constants.py        # 座標、顏色、路徑常數
├── data/
│   ├── board.json          # 48 格地產定義
│   └── items.json          # 5 種道具定義
├── assets/
│   ├── Map.png             # 棋盤圖
│   ├── Player1–4.png       # 角色圖示
│   └── Background.mp3      # 背景音樂
└── test_integration.py     # 整合測試
```

## 環境需求

- Python 3.10 以上
- pygame 2.0 以上

安裝依賴：

```bash
pip install pygame
```

## 啟動遊戲

```bash
cd monopoly-python
python main.py
```

## 操作方式

| 操作 | 說明 |
|------|------|
| 點擊畫面右下角左側區域 | 擲骰子，移動玩家 |
| 點擊畫面右下角右側區域 | 開啟道具欄 |
| 對話框「是」/ 「否」 | 確認購買或升級地產 |
| 商店視窗「購買」 | 購買道具 |
| 卡片彈窗 | 點擊或等待 2 秒繼續 |

## 遊戲規則

### 基本設定
- 玩家人數：4 人（布魯、瑞德、椰柔、古林）
- 起始金錢：$100,000
- 棋盤格數：48 格
- 過起點獎勵：+$2,000

### 地格類型

| 類型 | 說明 |
|------|------|
| 一般土地 | 可購買；踩到他人地產需付租金（地產價值 × 40%）|
| 機會 | 抽機會卡（共 20 張，洗牌循環）|
| 命運 | 抽命運卡（共 19 張，洗牌循環）|
| 商店 | 開啟商店，可購買道具 |
| 無功能 | 起點、瀑布、空地等，落地無事 |

### 地產系統
- 購買後可繼續升級（最高 4 級），升級費用為購入價的 60%
- 每升一級，租金基準增加
- 踩到格 29（瀑布上游）會被傳送至格 15（瀑布下游）

### 特殊區域
- **醫院**：被命運卡送進後，下一輪自動離開
- **監獄**：被命運卡送進後，下一輪自動離開；持有免獄卡可免疫入獄

### 道具一覽

| 道具 | 價格 | 效果 |
|------|------|------|
| 骰子 | $1,300 | 主動使用，額外擲一顆骰子（+1~6 步）|
| 健保卡 | $1,500 | 被動，進醫院時獲得 $1,800 補助 |
| 免獄卡 | $1,800 | 被動，入獄時直接免疫 |
| 房產VIP卡 | $900 | 被動，下次蓋房減免 $1,000 |
| 恭喜出獄卡 | $1,300 | 被動，入獄時獲得 $1,500 補助 |

### 破產判定
金錢 ≤ 0 時該玩家宣告破產，遊戲結束。
