# 大富翁

大富翁遊戲的 Python 版。

---

## 專案結構

```
monopoly-python/
├── main.py                 # 遊戲進入點
├── core/                   # 核心遊戲邏輯（無 UI 依賴）
│   ├── dice.py             # 骰子
│   ├── land.py             # 地產模型
│   ├── player.py           # 玩家狀態與移動
│   ├── chance_card.py      # 機會卡（20 張）
│   ├── fate_card.py        # 命運卡（19 張）
│   ├── shop.py             # 商店道具
│   └── game_state.py       # 回合狀態引擎
├── ui/                     # Pygame 畫面層
│   ├── board_renderer.py   # 棋盤與玩家精靈
│   ├── info_panel.py       # 右側訊息欄
│   ├── dialog.py           # 確認對話框 / 道具欄
│   ├── shop_screen.py      # 商店視窗
│   ├── card_popup.py       # 機會 / 命運卡彈窗
│   └── constants.py        # 座標、顏色、路徑常數
├── data/
│   ├── board.json          # 48 格地產定義
│   └── items.json          # 5 種道具定義
├── assets/
│   ├── Map.png             # 棋盤圖
│   ├── Player1–4.png       # 角色圖示
│   ├── houses/             # 4 位玩家的 1–4 級房屋圖示
│   └── Background.mp3      # 背景音樂
└── test_integration.py     # 整合測試
```

## 環境需求

- Python 3.10 以上
- pygame 2.0 以上

安裝依賴：

```bash
pip install pygame
```

## 啟動遊戲

```bash
cd monopoly-python
python main.py
```

## 操作方式

| 操作 | 說明 |
|------|------|
| 點擊畫面右下角左側區域 | 擲骰子，移動玩家 |
| 點擊畫面右下角右側區域 | 開啟道具欄 |
| 對話框「是」/ 「否」 | 確認購買或升級地產 |
| 商店視窗「購買」 | 購買道具；每次進商店只能買 1 件 |
| 卡片彈窗 | 點擊或等待 2 秒繼續 |
| 右側訊息欄滾輪 | 查看較早的遊戲紀錄 |

## 遊戲規則

### 基本設定
- 玩家人數：4 人（布魯、瑞德、椰柔、古林）
- 起始金錢：$100,000
- 主棋盤格數：48 格；另有醫院路線 48–52、監獄路線 53–56
- 過起點獎勵：+$2,000
- 同一格有多位玩家時，角色會分散在左上、右上、左下、右下顯示

### 地格類型

| 類型 | 說明 |
|------|------|
| 一般土地 | 可購買；踩到他人地產需付租金（地產價值 × 40%）|
| 機會 | 抽機會卡（共 20 張，洗牌循環）|
| 命運 | 抽命運卡（共 19 張，洗牌循環）|
| 商店 | 開啟商店，可購買道具 |
| 無功能 | 起點、瀑布、空地等，落地無事 |

### 地產系統
- 購買後可繼續升級（最高 4 級），升級費用為購入價的 60%
- 每升一級，租金基準增加
- 地圖會依玩家與等級顯示房屋圖示（1–4 級）
- 部分大型地產佔兩格，踩到其中任一格都算同一塊地（例如格 13/14、27/28）
- 踩到格 29（瀑布上游）會被傳送至格 15（瀑布下游）

### 特殊區域
- **醫院**：命運卡可能將玩家送到醫院路線（位置 52）。之後擲骰會沿醫院路線前進，超過 52 後回到主棋盤 45 附近。
- **監獄**：命運卡可能將玩家送到監獄路線（位置 56）。之後擲骰會沿監獄路線前進，超過 56 後回到主棋盤 12 附近。
- **停一回合**：目前主要由機會卡的休息效果造成。若玩家 `stop_round > 0`，輪到該玩家時會顯示跳過回合提示並自動換下一位。
- 進醫院或監獄本身不會自動停一回合；必須同時有停回合狀態才會跳過。

### 道具一覽

| 道具 | 價格 | 效果 |
|------|------|------|
| 骰子 | $1,300 | 主動使用，額外擲一顆骰子（+1~6 步）|
| 健保卡 | $1,500 | 被動，進醫院時獲得 $1,800 補助 |
| 免獄卡 | $1,800 | 被動，入獄時直接免疫 |
| 房產VIP卡 | $900 | 被動，下次蓋房減免 $1,000 |
| 恭喜出獄卡 | $1,300 | 被動，入獄時獲得 $1,500 補助 |

### 破產判定
金錢 ≤ 0 時該玩家宣告破產，遊戲結束。

### UI 回饋
- 商店購買成功後，已購買的道具列會高亮，其他商品會鎖定。
- 金錢變化會在玩家資訊卡上以紅色 / 綠色徽章短暫顯示。
- 擲骰移動造成的付租、收租、卡片金錢效果，會等角色抵達定點後才顯示金錢變化。
- 右側訊息欄保留較多歷史紀錄，畫面顯示約 20 行，可用滑鼠滾輪回看。

### 卡片概況
- 機會卡共 20 張，包含金錢增減、免費升級房屋、休息一回合等效果。
- 命運卡共 19 張，包含金錢增減、前進 / 後退、送醫院、送監獄、取得免獄卡、拆房等效果。
- 免費升級房屋不會超過 4 級。
